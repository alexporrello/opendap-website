OPeNDAP Software

Hyrax

The next generation server from OPeNDAP.

Hyrax uses a modular design that employs a lightweight Java servlet to provide the public-accessible client interface, and a back-end daemon, the BES, to handle the heavy lifting. The BES uses the same handlers that are used with Server3 (also know as the CGI Server) but loads those at run time.

Libdap

C++ SDK that contains an implementation of DAP 2.0 and DAP4.0

The libdap includes both Client- and Server-side support classes. Our software distributions are signed using a public/private key pair.
